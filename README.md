# CSE 5234 — Lab 1 (Route A): Node.js Starter

This repo is a *minimal* Node.js project you can publish to GitHub from VS Code.

## Quick Start
1. Open a terminal in this folder and run:
   ```bash
   node -v
   npm -v
   npm start
   ```
   You should see: `Hello CSE 5234! Your Node.js environment is working.`

2. In VS Code: **Source Control** → **Publish to GitHub** (or use the CLI below).

## Optional: Git CLI
```bash
git init
git add .
git commit -m "Lab 1: Node route starter"
git branch -M main
git remote add origin https://github.com/<YOUR_USER>/hello-node-lab1.git
git push -u origin main
```
